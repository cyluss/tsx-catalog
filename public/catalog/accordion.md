# Accordion

<dl>
  <dt>원칙</dt>
  <dd>콘텐츠를 접어 공간을 절약한다. 모든 항목을 동시에 볼 필요가 없을 때 사용한다.</dd>
  <dt>사용 시점</dt>
  <dd>
    <div class="do-dont">
      <div class="do"><strong>써야 할 때</strong><p>FAQ — 질문 목록</p><p>설정 화면 — 카테고리별 옵션 그룹</p><p>상세 정보 — 기본 정보 외 추가 항목</p></div>
      <div class="dont"><strong>쓰지 말아야 할 때</strong><p>항목이 2개 이하 — 단순 토글 사용</p><p>모든 내용을 항상 봐야 할 때</p><p>탭으로 구분 가능한 동일 수준 콘텐츠 — Tabs 사용</p></div>
    </div>
  </dd>
  <dt>선택 규칙</dt>
  <dd>기본은 단일 열기(하나 열면 나머지 닫힘). 사용자가 여러 항목을 비교해야 한다면 <code>alwaysOpen</code> (Bootstrap) / <code>multiple</code> (shadcn) 사용.</dd>
</dl>

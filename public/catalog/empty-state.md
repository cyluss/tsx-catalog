# Empty State

<dl>
  <dt>원칙</dt>
  <dd>데이터가 없는 상태를 빈 화면으로 방치하지 않는다. 이유를 설명하고 다음 행동을 안내한다.</dd>
  <dt>사용 시점</dt>
  <dd>
    <div class="do-dont">
      <div class="do"><strong>써야 할 때</strong><p>목록에 항목이 없을 때</p><p>검색 결과가 없을 때</p><p>처음 사용하는 화면</p></div>
      <div class="dont"><strong>쓰지 말아야 할 때</strong><p>로딩 중 — Spinner 사용</p><p>에러 상태 — Alert 사용</p></div>
    </div>
  </dd>
  <dt>구성 요소</dt>
  <dd>아이콘(선택) + 제목(필수) + 설명(선택) + 액션 버튼(선택). 액션이 없으면 버튼을 생략한다.</dd>
  <dt>선택 규칙</dt>
  <dd>Table, 목록 컴포넌트와 함께 사용한다. 단독 페이지 전체에 표시할 때는 세로 중앙 정렬한다.</dd>
</dl>
